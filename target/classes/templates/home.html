원하는 사용자 목록, 토큰 생성, Swagger 또는 Actuator를 선택하세요.

사용자 관리:

  • 사용자목록(DB1) - DB1의 사용자만 조회/관리
  • 사용자목록(DB2) - DB2의 사용자만 조회/관리
  • 통합사용자관리 - 두 DB에 동시에 등록/수정/삭제

관리자 기능:

  • 토큰 생성 - 관리자 토큰 생성
  • Swagger UI - REST API 문서

시스템 정보:

  • Health Check - 시스템 상태 확인
  • REST Health Check - REST API 상태 확인

REST API 엔드포인트:

사용자 관리 API:

  • DB1 사용자: /api/users/db1
  • DB2 사용자: /api/users/db2
  • 통합 사용자: /api/users/sync
  • 기본 사용자: /api/users

인증 API:

  • 로그인: POST /api/auth/login
  • 로그아웃: POST /api/auth/logout
  • 상태 확인: GET /api/auth/status

관리자 API:

  • 토큰 생성: POST /api/admin/token/generate
  • 토큰 상태: GET /api/admin/token/status

시스템 API:

  • Health Check: GET /api/health

참고: 모든 REST API는 JSON 형식으로 요청/응답하며, Swagger UI에서 상세한 API 문서를 확인할 수 있습니다.
