🔄 통합 사용자 등록 (DB1 + DB2)

※ 이 폼에서 등록하면 두 DB에 동시에 저장되며, CommonDTO 정보도 함께 기록됩니다.

🧑‍💼 기본 사용자 정보

🏦 거래 시스템 정보 (CommonDTO)

사용자 등록과 함께 기록되는 거래 시스템 공통 정보입니다.

취소

🚀 통합 사용자 등록 특징:

  • 동시 저장: DB1과 DB2에 동시에 사용자 정보가 저장됩니다.
  • CommonDTO 통합: 사용자 등록과 함께 거래 시스템 정보도 기록됩니다.
  • 일관성 보장: 두 DB 간의 데이터 일관성이 자동으로 유지됩니다.
  • 롤백 처리: 한쪽 DB에서 오류 발생 시 전체 트랜잭션이 롤백됩니다.
  • 필수 항목(*): 반드시 입력해야 하는 항목입니다.

⚠️ 주의사항:

  • 사용자명은 고유해야 합니다.
  • 비밀번호는 최소 6자 이상이어야 합니다.
  • 나이는 0-120 사이의 값이어야 합니다.
  • IP 주소는 IPv4 형식으로 입력해주세요.
  • 은행/지점 코드는 3자리 숫자로 입력해주세요.
